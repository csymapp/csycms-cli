# csycms CLI

A CLI for csycms. CSYCMS is a flat file content management sytem for nodejs.

# Dependancies

 - nginx
 - certbot
 - nodejs

Please make sure the dependencies are installed before proceeding.

# Installing

```
npm install -g csycms
```


# initializing

Should start service

```
sudo csycms init
```

# Starting service

```
sudo csycms start
```
# Stopping service

```
sudo csycms stop
```
# Starting restarting

```
sudo csycms restart
```
